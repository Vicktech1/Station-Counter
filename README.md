
# 🚉 Station Counter

The **Station Counter** is a simple JavaScript web application that counts the number of people entering a station and allows the user to save previous entries. This project demonstrates basic **HTML, CSS, and JavaScript concepts** such as DOM manipulation, event handling, and updating elements dynamically.

---

## 📌 Project Overview

The Station Counter app allows users to:

* Increment the number of people entering the station.
* Save the current count.
* Display previously saved counts.

This project is useful for beginners learning how to connect **JavaScript functions with HTML buttons** and update content on a webpage.

## 📄 HTML Structure

The HTML file creates the user interface of the application.

Main elements include:

* A **heading** for the application
* A **counter display**
* An **Increment button** to increase the count
* A **Save button** to store previous entries
* A section showing saved entries

## ⚙️ How It Works

### Increment Button

When the **INCREMENT** button is clicked, the `increment()` function increases the count by **1** and updates the displayed number.

### Save Button

When the **SAVE** button is clicked, the `save()` function:

1. Stores the current count.
2. Displays it in the **Previous entries** section.
3. Resets the counter to **0**.

## ▶️ How to Run the Project

1. Download or clone the project.
2. Open the folder in **Visual Studio Code**.
3. Open the `index.html` file.
4. Run it in a browser.

You will see the **Station Counter interface** where you can increment and save counts.
